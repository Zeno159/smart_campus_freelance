# Smart Campus Freelance Marketplace

## Overview

A full-stack web platform that enables students to post freelance services, apply for campus jobs, manage contracts, and securely interact within a university ecosystem.

The project provides authentication, profile management, job listings, applications, digital contracts, and wallet functionality through a FastAPI backend with a responsive frontend.

## Features

- Student Authentication
- Profile Management
- Job Posting
- Job Applications
- Service Listings
- Contract Management
- Wallet Module
- REST API Architecture
